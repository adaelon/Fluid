**测试**(确定性 B2):后端 `cargo test` **33/33 绿**(rfd 集成后整体编译通过;`pick_folder` 弹 GUI 对话框、本质交互,无法自动测 —— 其下游换根已由 U3 的 `root_swap_*` 单测覆盖)。前端 `npm run build` 绿,CSS 5.92kB。**未自动验证**(A2 诚实):点按钮→系统对话框真实弹出→选中→换根的端到端交互 —— GUI 交互 + 需真实桌面,本环境沙箱无,留用户眼验。
